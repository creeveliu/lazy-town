# AGENTS

## 项目目标

构建一个极简风格的“热门游戏发售表 + 院线电影上映表 + 在线电影上线表”网站。

## 当前业务规则

- 展示游戏、院线电影、在线电影三个 tab
- 游戏数据来源：第三方游戏发售表
- 院线电影数据来源：第三方电影上映表
- 在线电影数据来源：爱奇艺 / 腾讯视频 / 优酷等流媒体平台
- 游戏平台：PC / PS5 / Switch
- 游戏时间范围：未来 1 年
- 游戏热度阈值：`> 4000`
- 在线电影预约阈值：`>= 5000`
- 排序规则：按发售/上映/上线日期升序

## UI 约束

- 极简风格，信息密度高但不花哨
- 主体为 tab + 表格视图
- 海报可展示但尺寸小
- 不显示热度数字，仅显示标签：`普通 / 热门 / 爆款`

## 交互约束

- 不提供额外排序器
- 游戏名、院线电影名、在线电影名可点击跳转详情页
- 移动端优先保证可读性，避免横向溢出

## 工程约束

- 尽量保持服务端抓取逻辑简单直接
- 变更优先最小改动
- 每次改动后至少执行一次 `npm run lint`
